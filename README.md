# Price Check — Portfolio Case Study

A presentation walking through how Vendr turned $3B+ in locked SaaS transaction data into a self-service pricing intelligence product and a compounding data moat.

## Project

**Price Check** lets SaaS buyers upload a contract and get an AI-powered pricing analysis in under a minute — benchmarked against real transaction data from 20,000+ products.

This case study covers: problem definition, persona, strategy, the phased build (concierge experiment → automation → public launch → viral share), challenges, results, and reflections.
